## 🐛 常见问题

### ❓ "面试无法启动"
**检查：**
1. 是否上传简历或输入简历内容
2. 浏览器 Console 是否有错误
3. 网络连接是否正常

### ❓ "API 响应很慢"
**正常情况：**
- 首次调用 1-2 秒
- 后续调用 2-3 秒

**如果超过 5 秒，检查：**
- 网络连接稳定性
- 豆包官网是否可访问
- 简历文本是否过长 (>5000 字符)

### ❓ "回答为空或出现错误"
**解决：**
1. 打开浏览器 DevTools (F12)
2. 查看 Console 中的错误信息
3. 检查是否禁用了浏览器跨域请求

### ❓ "自动降级到模拟数据"
**原因：**
- 豆包 API 暂时不可用
- API 配额限制
- 网络错误

**结果：**
- UI 继续正常工作
- 使用预定义的模拟回答
- 刷新页面重新尝试

## 💡 技巧

### 优化面试效果

**简历应包含：**
✅ 教育背景（学校、专业、年级）  
✅ 科研/项目经验（2-3 个）  
✅ 获奖或特殊经历  
✅ 未来研究方向

**面试类型选择：**
- 夏令营 (camp) = 宽泛基础问题
- 笔试面试 (interview) = 深度专业问题

"我的项目是关于绿色建筑设计的，涉及从初期规划到最终实现的全过程..."

"我做过一些项目..."
